My Twitter Timelines is all in one twitter widget. With this one widget you can display twitter user timelines, user favourites, Search timeline, List timeline, Collection timeline. You don't have to install multiple twitter widget any more to manage your twitter feed on website. Just install My Widget and Enjoy.

== Description ==

Demo :: http://audaciotech.com/wordpress/documentation/

My Twitter Timelines is a suite of embeddable tools to integrate Twitter and display Tweets in your wordpress site. By using our widget you can display your own twitter feeds on your website.

My Twitter Timelines don’t have only one feature, this widget can be proved as your complete twitter solution.
By using this widget you can display various timelines at a time. Those are:
1.	User Timelines
2.	User Favorite Timelines
3.	Search Timeline
4.	List Timeline
5.	Collection Timeline

Highly Customizable::
Not only the widget have many options to display twitter timelines, It also have many customizable options in backend. You can control over many twitter settings from backend.
We have included nice documentation to make sure that you able to make the widget work properly on your website.

We love helping people and you can ask us any questions. We will love to reply.

== Installation ==

1. Upload plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. go to Widgets page and Enjoy our Plugin.

For more details about widget settings go here - 

http://audaciotech.com/wordpress/documentation/
